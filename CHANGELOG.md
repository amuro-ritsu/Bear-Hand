# 変更履歴

## [v1.0.0] - 2024-12-05

### ✨ 新機能（Initial Release - ブラウザ版）

#### コア機能
- マウス/ペンタブの動きを追跡
- カーソル位置に連動した手の移動アニメーション
- **ペン先を中心とした円運動アニメーション**（改良版）
- 画像ファイルの相対パス読み込み
- URLパラメータによる画像指定

#### 画像管理
- **HTMLと同じフォルダから画像を読み込み**
- hand.png / tablet.png のデフォルト対応
- URLパラメータで別の画像も指定可能
- プリセット機能（複数設定を保存）

#### UI/UX
- 直感的な設定パネル
- リアルタイムプレビュー
- スライダーによる細かい調整
- レスポンシブデザイン
- 初期セットアップ画面

#### 設定機能
- 感度調整（0.1〜3.0）
- スムージング ON/OFF
- スムージング強度（0.1〜0.9）
- ペンの傾き角度（0〜45度）
- ペンの揺れ速度（1〜30）
- X/Y座標オフセット
- 個別の画像サイズ調整
- タブレット位置調整
- 表示/非表示切り替え
- **画面端制限機能**（左右上下）

#### 技術的特徴
- 完全透過背景対応
- 60 FPS動作
- 低CPU/メモリ使用率
- クラッシュしにくい安定動作
- LocalStorage設定保存
- URLパラメータによる柔軟な設定

### 🎯 動作環境
- Windows 10/11
- macOS 10.15以降
- OBS Studio v27.0以降
- Chrome/Edge/Firefox（最新版）

### 📝 ドキュメント
- README.md（詳細マニュアル）
- QUICKSTART.md（3分スタートガイド）
- IMAGE_SETUP.md（画像配置ガイド）★NEW
- LICENSE.txt（MITライセンス）
- samples/README.md（サンプル画像の入手方法）★NEW

### 🔧 改善点

#### v1.0.0 最終版
- ペンの揺れを「回転」から「円運動」に変更（よりリアルに）
- 画面端制限機能を追加（左右上下個別設定可能）
- 画像読み込み方式を相対パスに変更（OBS対応）
- URLパラメータで画像ファイルを指定可能に
- プリセット機能で複数設定を管理可能に
