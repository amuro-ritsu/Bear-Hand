# 変更履歴

## [v1.0.0] - 2024-12-05

### ✨ 新機能（Initial Release - ブラウザ版）

#### コア機能
- マウス/ペンタブの動きを追跡
- カーソル位置に連動した手の移動アニメーション
- マウスボタン押下時のペン揺れアニメーション
- 画像アップロード機能（手+ペン、タブレット）
- LocalStorageによる設定・画像の保存

#### UI/UX
- 直感的な設定パネル
- リアルタイムプレビュー
- スライダーによる細かい調整
- レスポンシブデザイン
- 初期セットアップ画面

#### 設定機能
- 感度調整（0.1〜3.0）
- スムージング ON/OFF
- スムージング強度（0.1〜0.9）
- ペンの傾き角度（0〜45度）
- ペンの揺れ速度（1〜30）
- X/Y座標オフセット
- 個別の画像サイズ調整
- タブレット位置調整
- 表示/非表示切り替え

#### 技術的特徴
- 完全透過背景対応
- 60 FPS動作
- 低CPU/メモリ使用率
- クラッシュしにくい安定動作
- LocalStorage自動保存

### 🎯 動作環境
- Windows 10/11
- macOS 10.15以降
- OBS Studio v27.0以降
- Chrome/Edge/Firefox（最新版）

- LICENSE.txt（MITライセンス）
