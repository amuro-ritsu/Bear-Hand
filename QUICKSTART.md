# クイックスタートガイド
## 3分で始めるお絵かき手元アニメーション（ブラウザ版）

このガイドに従えば、すぐにお絵かき配信で手元アニメーションが使えます！

---

## 📦 必要なもの

✅ OBS Studio（インストール済み）
✅ drawing_hand_animator.html
✅ ペンを持った手の透過PNG画像（後からでもOK）
✅ タブレットの透過PNG画像（後からでもOK）

---

## 🚀 ステップ1: OBSにブラウザソースを追加（1分）

### 1-1. ブラウザソースを追加
```
OBS Studio → ソース → ＋ボタン → ブラウザ
```

### 1-2. 名前を設定
```
名前: お絵かき手元
（好きな名前でOK）
```

### 1-3. ファイルを指定
```
✅ ローカルファイル: チェックを入れる
□ 参照: drawing_hand_animator.html を選択
```

### 1-4. サイズを設定
```
幅: 1920
高さ: 1080
```

### 1-5. その他の設定
```
✅ シャットダウン時にブラウザを更新: チェック
✅ シーンがアクティブになったとき更新: チェック
□ FPS: 60
```

### 1-6. カスタムCSSを設定（重要！）
```
カスタムCSS欄に以下をコピペ:

body { 
    background-color: rgba(0, 0, 0, 0) !important; 
    margin: 0px auto;
    overflow: hidden;
}
```

これで背景が透過されます。

### 1-7. OKをクリック

---

## 🎨 ステップ2: 画像をアップロード（1分）

### 2-1. 初期画面
ブラウザソースを追加すると、ウェルカム画面が表示されます。

```
「開始する」ボタンをクリック
```

### 2-2. 設定パネルを開く
```
右上の「⚙️ 設定」ボタンをクリック
```

### 2-3. 手の画像をアップロード
```
📁 画像設定 セクション

「手+ペンの画像」の
「ファイルを選択」をクリック
→ 用意した手の画像（PNG）を選択
```

プレビューが表示されればOK！

### 2-4. タブレット画像をアップロード
```
「タブレット画像」の
「ファイルを選択」をクリック
→ 用意したタブレット画像（PNG）を選択
```

### 2-5. 設定を保存
```
一番下の「設定を保存」ボタンをクリック
```

---

## ✅ ステップ3: 動作確認（30秒）

### 3-1. プレビューで確認
```
OBSのプレビュー画面内でマウスを動かす
→ 手とタブレットが動けば成功！
```

### 3-2. 描画テスト
```
左クリックを押したまま動かす
→ ペンが左右に揺れれば完璧！
```

## 🎬 おまけ: レイヤー順序の調整

### 理想的なソース順序

OBSのソース欄で上から順に：

```
1. VTuberモデル         ← 一番上
2. お絵かき手元         ← このブラウザソース
3. お絵かきソフト        ← ウィンドウキャプチャ
4. その他のソース
5. 背景                ← 一番下
```

ドラッグ＆ドロップで順序変更可能。

---

## 🔧 簡単な調整

### 手の位置がずれている場合

```
設定パネル:
📐 位置調整 セクション
├─ X座標オフセット: 左右に調整
└─ Y座標オフセット: 上下に調整
```

### 動きが速すぎる/遅すぎる場合

```
設定パネル:
🎮 動作設定 セクション
└─ 感度: 1.0 → 0.5（遅く） or 1.5（速く）
```

### ペンの揺れを調整したい場合

```
設定パネル:
✏️ ペンアニメーション セクション
├─ ペンの傾き角度: 15 → 20（大きく揺れる）
└─ ペンの揺れ速度: 10 → 15（速く揺れる）
```

### 画像サイズを変更したい場合

```
設定パネル:
📐 位置調整 セクション
├─ 手の画像サイズ: 1.0x → 1.5x（大きく）
└─ タブレット画像サイズ: 1.0x → 1.2x（大きく）
```

## 📸 画像がない場合

### 今すぐ試したい方へ

画像なしでも動作確認は可能です：

1. OBSにブラウザソースを追加
2. 「開始する」をクリック
3. 設定パネルで各パラメータを調整
4. マウスの動きに合わせて設定が変わるか確認

画像は後からアップロードできます！

### 画像の入手方法

#### 方法1: 自分で描く
- CLIP STUDIO PAINT
- Photoshop
- Krita（無料）
などで手とタブレットを描く

#### 方法2: フリー素材を使う
- いらすとや
- Pixabay
などで「hand holding pen」を検索

#### 方法3: AI生成
- Stable Diffusion
- Midjourney
などで生成

詳しくは [IMAGE_GUIDE.md](IMAGE_GUIDE.md) を参照。
